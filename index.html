Video Object Tracking

As our final project in the course Computer Vision, I implemented the mean shift tracking algorithm. This tracker is histogram-based and utilizes the statistics of a region, which makes it well suited for tracking deformable objects. The following video is the result of setting the tracker to track the object that lies at [ , ], the coordinates of the green cup. This MATLAB program redraws the bounding box for the object at every frame.

Tailor'd

A fashion mobile application that won the category for “Best Mobile App” at HackCMU. Using complementary color palettes, Tailor’d can randomly generate coordinated outfits and let you keep an inventory of your current wardrobe. Built using Python, HTML, CSS, and Javascript in the Ionic mobile application and Flask frameworks, this application synthesizes clean design with an intuitive user interface. Collaboration with Kristin Yin, Selena When, and Tiffany Zhu.

3D Aquarium

The 3D Aquarium simulator was my final project for the course “Fundamentals of Programming and Computer Science.” The course was taught entirely in Python, but I decided to explore and use new technologies such as Panda3D (3D game engine) and Blender (computer graphics software). In this simulator, one can add fish to the aquarium and have the fish simulate community and aggressive behaviors. Feeding and user controls were also incorporated into the simulator.
